# 레이아웃 스니펫 모음 (복붙해서 쓰세요)

> 아래 블록을 `slides.md`에 그대로 붙여넣고 내용만 바꾸면 됩니다.
> 슬라이드 사이는 항상 빈 줄 + `---` + 빈 줄로 구분합니다.

## 카드 그리드 (마우스 호버 효과)

```
^ 라벨
## 슬라이드 제목

::: cards
### 라벨1
**제목1**
설명 한 줄
+++
### 라벨2
**제목2**
설명 한 줄
+++
### 라벨3
**제목3**
설명 한 줄
:::
```
> 카드는 자동으로 가로 정렬되고, 마우스를 올리면 테두리 강조·살짝 떠오릅니다. (마우스 따라오는 글로우는 모든 슬라이드에 자동)

## 막대 차트

```
^ 차트
## 채널별 비중

::: bars
모바일 | 68%
데스크톱 | 24%
태블릿 | 8%
:::
```
> `라벨 | 값` 의 숫자로 막대 길이가 자동 계산됩니다.

## 흐름도 / 파이프라인

```
^ 흐름
## 자동화 4단계

::: flow
이벤트 감지 → 경계 결정 → 자동 편집 → 자동 배포
:::
```
> `→` 로 단계를 나누면 박스+화살표로 그려집니다.
