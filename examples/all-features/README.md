# 예제 — 전체 기능 샘플 (3장)

`lecture-deck`의 주요 기능을 3장에 담은 데모입니다.

![데모](../../docs/demo.gif)

## 바로 보기
- **`deck.html`** 다운로드 후 더블클릭 → 서버 없이 바로 열립니다(자립형).
- 소스는 **`slides.md`** 한 파일. 이걸 보고 문법을 익히세요.

## 3장에 담은 것

| 슬라이드 | 보여주는 기능 |
|---|---|
| **1 · 표지** | `<!-- title -->` 표지 · `^` kicker 라벨 · `*기울임*`·`**굵게**` · `???` 발표자 노트 |
| **2 · 카드** | `::: cards` 카드 그리드 + **마우스 호버 효과** · `@img` 이미지 자리 |
| **3 · 데이터** | `::: bars` 막대 차트 · `::: flow` 흐름도 · `@src` 출처 표기 |

**모든 슬라이드 자동 적용**: 마우스 클릭/우클릭·리모컨 넘기기 · 커서 글로우 · 단축키(`F` 전체화면 · `O` 한눈에 · `S` 노트 · `P` 인쇄 · `?` 도움말).

## 직접 빌드하려면
스킬 툴킷(`skills/lecture-deck/template/`)을 복사한 폴더에 이 `slides.md`를 넣고:
```bash
python3 build.py     # deck.html 생성
python3 lint.py      # 자가 점검(이 예제는 경고 0)
```

> 전면 레이아웃(`<!-- section -->`·`<!-- dark -->`·`<!-- center -->`)과 `::` 큰 숫자·표·인용은 각각 한 장을 쓰는 요소라 이 3장 샘플에는 뺐습니다. 전체 문법은 `skills/lecture-deck/template/_snippets/레이아웃모음.md` 참고.
